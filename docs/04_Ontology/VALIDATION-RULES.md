# VALIDATION-RULES

## Generales

1. ID único.
2. Tipo reconocido.
3. Estado válido.
4. Fuente identificada.
5. Título no vacío.
6. Enlaces existentes o marcados como pendientes.
7. Sin duplicados semánticos.

## Clínicas

1. Revisión humana.
2. Diferenciar hallazgo, diagnóstico e hipótesis.
3. Evitar indicaciones universales.
4. Registrar banderas rojas cuando proceda.
5. Registrar incertidumbre.
6. No inventar bibliografía ni resultados.

## Medios y SEO

1. URL original.
2. Medio identificado.
3. Fecha cuando esté disponible.
4. Diferenciar contenido propio, patrocinado y cobertura independiente.
