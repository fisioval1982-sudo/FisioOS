# ASSET-REVIEW-WORKFLOW

## Flujo

Draft
→ Clinical Review
→ Visual Review
→ Technical Review
→ Approved
→ Published
→ Deprecated

## Clinical Review

Valida:

- estructura;
- lateridad;
- orientación;
- proporciones;
- hitos anatómicos;
- relaciones espaciales.

## Visual Review

Valida:

- estilo;
- color;
- iluminación;
- fondo;
- halos;
- encuadre;
- consistencia de colección.

## Technical Review

Valida:

- formato;
- resolución;
- nombre;
- ruta;
- metadatos;
- enlaces;
- versión.

## Aprobación condicionada

Un activo puede aprobarse con limitaciones técnicas, pero la limitación debe quedar declarada en su ficha.
