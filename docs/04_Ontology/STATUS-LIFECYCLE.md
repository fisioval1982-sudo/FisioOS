# STATUS-LIFECYCLE

```text
Candidate → Draft → Review → Approved → Deprecated
```

- Candidate: importado o detectado sin revisión.
- Draft: entidad en construcción.
- Review: contenido completo pendiente de validación.
- Approved: validado e incorporado.
- Deprecated: no debe utilizarse en nuevos contenidos.

## Reglas

- Contenido clínico requiere revisor humano.
- Las correcciones mayores incrementan versión.
- Una entidad deprecada no se elimina si tiene enlaces.
