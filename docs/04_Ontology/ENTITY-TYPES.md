# ENTITY-TYPES

| Prefijo | Tipo | Descripción |
|---|---|---|
| PERSON | Persona | Profesional, investigador o colaborador |
| ORG | Organización | Empresa, universidad, hospital, medio o institución |
| PROFILE | Perfil externo | ORCID, Scholar, Doctoralia, GBP, etc. |
| PAT | Patología | Entidad clínica o nodo maestro de patología |
| STR | Estructura | Anatomía y regiones |
| FIND | Hallazgo | Hallazgo clínico, ecográfico o funcional |
| TEST | Test | Prueba clínica o funcional |
| TRT | Tratamiento | Intervención terapéutica |
| EX | Ejercicio | Ejercicio terapéutico individual |
| PROGRAM | Programa | Conjunto estructurado de ejercicios |
| PRO | Protocolo | Procedimiento clínico u operativo |
| CASE | Caso | Caso clínico anonimizado |
| PUB | Publicación | Artículo científico |
| THESIS | Tesis | Tesis doctoral |
| RESEARCH | Línea de investigación | Agrupación temática |
| MEDIA | Aparición en medios | Noticia, entrevista, radio o televisión |
| DOSSIER | Dosier | Colección de apariciones |
| SERVICE | Servicio | Servicio prestado por FisioForYou |
| DATASET | Dataset | Conjunto de datos |
| ASSET | Activo | Imagen, vídeo, SVG, plantilla o material |
| PROCESS | Proceso | Secuencia humana o mixta |
| WORKFLOW | Workflow | Implementación operativa o automatizable |
| CAP | Capacidad | Función que FisioOS puede ejecutar |

## Regla

No se crearán nuevos tipos sin justificar su necesidad y actualizar este documento.
