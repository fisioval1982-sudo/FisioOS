# ID-STANDARD

## Formato

`PREFIJO-000001`

## Reglas

1. Seis dígitos.
2. El identificador no se reutiliza.
3. El título puede cambiar; el ID no.
4. Los sinónimos se guardan en `aliases`.
5. Un duplicado se fusiona manteniendo el ID más antiguo.
6. Un ID deprecado conserva una redirección.
7. Cambiar un prefijo es una migración excepcional.
