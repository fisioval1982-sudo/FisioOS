# MODULE-001 — Miembro superior

## Objetivo

Este módulo crea el primer bloque anatómico reutilizable de FisioOS correspondiente al miembro superior.

## Alcance

Incluye el nodo regional principal del miembro superior y las entidades anatómicas iniciales de hombro, brazo, codo, antebrazo, muñeca y mano.

## Entidades incluidas

- REG-000005 — Miembro superior
- STR-000100 — Hombro
- STR-000101 — Escápula
- STR-000102 — Clavícula
- STR-000103 — Húmero proximal
- STR-000104 — Articulación glenohumeral
- STR-000105 — Articulación acromioclavicular
- STR-000106 — Manguito rotador
- STR-000107 — Bursa subacromial-subdeltoidea
- STR-000108 — Corredera bicipital
- STR-000109 — Labrum glenoideo
- STR-000110 — Brazo
- STR-000120 — Codo
- STR-000130 — Antebrazo
- STR-000140 — Muñeca
- STR-000150 — Mano

## Criterio de validación

El módulo se considera válido cuando todas sus entidades aparecen en Obsidian, están versionadas en GitHub y enlazan jerárquicamente con REG-000001 o REG-000005.
