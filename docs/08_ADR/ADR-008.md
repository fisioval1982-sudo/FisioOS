# ADR-008 — Conservación de informes ecográficos e imágenes

## Decisión

**Sí.** Las imágenes ecográficas se conservarán como archivos independientes del PDF.

## Estructura obligatoria

CASE-000001/
- informe.pdf
- IMG-000001.jpg
- IMG-000002.jpg
- IMG-000003.jpg

## Reglas

1. El PDF se conserva como documento original.
2. Las imágenes se almacenan en JPG (o PNG si procede).
3. Cada imagen tendrá un identificador único.
4. Las imágenes serán la fuente para clasificación, IA y biblioteca ecográfica.
5. Los PDFs no serán la fuente principal para extraer imágenes.

## Consecuencia

Todos los nuevos casos deberán seguir esta estructura.
