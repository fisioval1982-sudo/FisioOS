# ADR-010 — Modelo oficial de ejercicios terapéuticos

## Decisión

Cada ejercicio terapéutico será una entidad `EX-XXXXXX` almacenada en `knowledge/exercise/`.

## Reglas

1. Los ejercicios externos entrarán como candidatos.
2. La validación clínica será obligatoria.
3. Un ejercicio no se asociará automáticamente a una patología.
4. La dosificación será contextual, no universal.
5. Las contraindicaciones y criterios de suspensión deberán registrarse.
6. Las imágenes o vídeos externos no se reutilizarán sin licencia compatible.
7. Las relaciones con patologías, protocolos y casos se expresarán mediante enlaces internos.
8. Se evitarán duplicados y sinónimos no normalizados.
