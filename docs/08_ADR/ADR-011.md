# ADR-011 — Adopción de Ontology System v1

## Decisión

FisioOS adopta Ontology System v1 como documento rector sin reorganizar la arquitectura actual.

## Consecuencias

- Se congelan los tipos y relaciones de la versión 1.
- Las nuevas entidades deberán cumplir los estándares.
- Capacidades y procesos se documentan aparte de la ontología clínica.
- Las migraciones deberán quedar registradas.
