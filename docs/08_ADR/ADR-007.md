# ADR-007 — Airtable como fuente operativa del conocimiento

## Decisión

Airtable será la fuente operativa única para:

- Entidades
- Relaciones
- Catálogos
- Bibliografía
- Estado de revisión

Obsidian contendrá la documentación generada automáticamente y GitHub será el repositorio oficial versionado.

## Flujo

Airtable → Scripts → Obsidian → GitHub

## Consecuencias

- No se editarán entidades directamente en Obsidian.
- Toda modificación comenzzará en Airtable.
- Cualquier cambio requerirá un nuevo ADR.
