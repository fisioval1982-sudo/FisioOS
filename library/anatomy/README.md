# Anatomy Library

Biblioteca de activos anatómicos reutilizables de FisioOS.

## Principios

- precisión anatómica;
- modularidad;
- lateridad explícita;
- vistas controladas;
- geometría maestra estable;
- prompts versionados;
- revisión clínica obligatoria.

## Rutas

- Fichas: `library/anatomy/assets/`
- Prompts: `library/anatomy/prompts/`
- Archivos: `assets/anatomy/`

## Colección inicial

La primera colección corresponde a la escápula derecha:

- vista posterior;
- vista anterior;
- vista lateral.

## Uso

Los activos maestros sirven como base para añadir:

- músculos;
- tendones;
- ligamentos;
- bursas;
- nervios;
- vasos;
- articulaciones;
- patología;
- señalización clínica.

No deben modificarse directamente.
