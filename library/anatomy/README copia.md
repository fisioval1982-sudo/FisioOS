# Anatomy Library

Biblioteca de activos anatómicos reutilizables de FisioOS.

## Ubicaciones

- Fichas: `library/anatomy/assets/`
- Prompts: `library/anatomy/prompts/`
- Archivos físicos: `assets/anatomy/`
- Estándares: `docs/04_Ontology/`
- Plantillas: `templates/`

## Principios

- anatomía validada;
- lateridad explícita;
- vistas ortográficas;
- geometría bloqueada tras aprobación;
- prompts versionados;
- relaciones entre masters y derivados;
- revisión clínica obligatoria.

## Colecciones activas

- escápula derecha;
- clavícula derecha.
