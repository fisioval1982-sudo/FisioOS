# FisioOS

Knowledge Operating System for Musculoskeletal Physiotherapy.

## Mission

Preserve, structure and reuse clinical knowledge generated by physiotherapists.

## Current status

- Version: 0.1
- Phase: Architecture
- Repository type: Docs as Code + Knowledge Engineering

## Entry point

Start here:

- `docs/00_Project/DOC-000.md`

## Repository structure

```text
docs/        Official documentation
knowledge/   Ontology, vocabulary and reusable clinical knowledge
corpus/      Episodes, cases, reports and structured exports
library/     Ultrasound images, annotations, SVG overlays and videos
research/    Papers, bibliography, protocols and guidelines
software/    Future SaaS, API, frontend, backend and AI modules
media/       General media assets
```
