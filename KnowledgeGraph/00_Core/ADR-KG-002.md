# ADR-KG-002 — Publicaciones y congresos como entidades

## Decisión

Los artículos científicos, pósteres, comunicaciones y congresos se incorporarán como entidades Markdown estructuradas.

- Publicaciones: `PUB-XXXXXX`
- Congresos y comunicaciones: `CONF-XXXXXX`
- El PDF será un adjunto, no la entidad principal.
