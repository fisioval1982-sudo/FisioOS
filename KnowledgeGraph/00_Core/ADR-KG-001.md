# ADR-KG-001 — Integración del Knowledge Graph dentro de FisioOS

## Decisión

El Knowledge Graph se integra dentro del mismo repositorio y la misma bóveda de Obsidian que FisioOS.

- `knowledge/`: conocimiento clínico.
- `docs/`: documentación, módulos y casos.
- `KnowledgeGraph/`: identidad, autoridad, perfiles, publicaciones, congresos, medios y servicios.
